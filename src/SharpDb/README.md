# SharpDb

**SharpDb** is a .NET library that provides a collection of extensions, helpers, and utilities to simplify and enhance database operations in your applications.

## Features
- **Extensions**: Additional methods to extend database functionality.
- **Entities**: Predefined entity classes for common database scenarios.
- **Services**: Dependency injection compatible classes that faciliate parts of data management logic.
- And more ...

## Requirements
- **Target Framework**: .NET 8 or higher
- **Dependencies**: None (but can be used alongside ORM libraries like Entity Framework Core)

## Installation
Either clone the repository and include the project in your solution, or install via NuGet.

## Usage
1. Add a reference to the library in your project.
2. Import the relevant namespaces to access the extensions, entities, and services.
3. Use the provided tools to simplify your database operations.

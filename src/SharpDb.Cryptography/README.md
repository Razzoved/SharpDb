# SharpDb.Cryptography

**SharpDb.Cryptography** is a .NET library that provides a collection of security extensions, helpers, and utilities for use with other SharpDb projects.

## Features
- Common hashing algorithms
- Determistic encryption and decryption utilities

## Requirements
- **Target Framework**: .NET 8 or higher
- **Dependencies**: None (but can be used alongside ORM libraries like Entity Framework Core)

## Installation
Either clone the repository and include the project in your solution, or install via NuGet.

## Usage
1. Add a reference to the library in your project.
2. Import the relevant namespaces to access the extensions, entities, and services.
3. Use the provided tools to simplify your database operations.
